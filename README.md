# burger_kiosk
Kiosk with Java swing GUI

# function
- Jtabbedpane 으로 메뉴 분류
- ArrayList<JPanel> 을 이용해 객체 list 사용
- 소켓 통신을 통한 주문 목록 전달
- 주문 목록 excel export
